常見比特幣術語介紹

  • Address

地址

比特幣地址由一連串字母和數字組成。你的錢包可以生成無限量的地址，每個地址都具有唯一性。

  • Bip

比特幣改進建議

比特幣改進建議，也規範如何提交、批准和被標記對協議的更改。

  • bitcoin (small ‘b’)

比特幣單位 (小寫的"b")

當您看到bitcoin的"b"為小寫時，表示為貨幣單位。

  • Block

區塊

將許多交易分組並由礦工進行挖掘。

  • Blockchain

區塊鍊

每一個成功挖掘的區塊組合而成的鍊(chain)，每個區塊前後都鍊接著另一個區塊。

  • Cold Storage

冷儲存

在沒有連上網際網路的環境中儲存比特幣私鑰的方式。

  • Confirmation

確認

當區塊中的其中一筆交易被成功挖掘時，即為一筆確認。隨後每一個新區塊的產生，都會讓這個交易地確認數加一。

  • Consensus Rules

共識規則

每一個參與比特幣網路的用戶都共同遵循的一組規則

  • Difficulty

挖礦難度

一個變動的值，用來指示目前開採新區塊所需的計算能力。

  • Fees

手續費

支付礦工將自己的交易納入開採區塊的費用(以比特幣計算)

  • Hardware Wallet

硬體錢包

用來在離線環境儲存比特幣私鑰專用的安全設備。

  • Hash

哈希

比特幣礦工處理交易時也會用到的一種數據轉換的動作。

  • Hash Rate

哈希值

礦工或採礦池執行的哈希總數，以哈希/秒表示。

  • KYC

實名認證

是一種要求銷售比特幣的業者必須獲得購買者身份訊息的規定。

  • Lightning Network

閃電網路

在比特幣之上構建的第二層支付協議，允許用戶以非常低的費用即時發送/接收小額付款。

  • Mempool

礦池

等待礦工將其放進區塊中的交易組合

  • Miner

礦工

一種提供處理交易的計算能力來賺取費用和獎勵的網絡參與者。

  • Multisig

多重簽名

多重簽名是一種需要多個私鑰來簽署交易的錢包設定方式。

  • Node

節點

一個連接到比特幣網絡的電腦，用來運行廣播，傳達和驗證交易的軟體。

  • Paper Wallet

紙錢包

是包含離線存儲比特幣所需的數據的一張紙，這應該避免被竊取。

  • Private Key

私鑰

用來儲存和交易比特幣的一組獨特字串，絕對不能和任何人分享。

  • Proof of Work

工作量證明

以礦工耗用的時間、裝置與能源做為成本，來保護比特幣網路安全和計算交易區塊的方法。

  • Public Key

公開金鑰

從私鑰產生，檢查交易簽名時會使用到。

  • Reward

獎勵

區塊獎勵會給予第一個取得足夠工作量證明的礦工。 獎勵 = 區塊補貼+該區塊所有的交易費用。目前的區塊補貼為6.25顆比特幣。

  • Satoshi or ‘sat’

聰

聰是比特幣的匿名創建者的名字，全名為中本聰(Satoshi Nakamoto)，有時縮寫為“sat”，是比特幣的最小單位。一個比特幣等於一億聰。

  • Satoshi Nakamoto

中本聰

比特幣的創造者（或者是一群創造者）從網路上消失前所使用的化名。

  • SegWit

隔離見證

隔離見證是2017年進行的一項改進，目的是提高比特幣網絡的可擴展性。

  • Subsidy

區塊補貼

區塊補貼支付給在每個區塊獲得足夠工作量證明的礦工。目前的補貼為6.25比特幣，而每21萬個區塊，補貼將減半（這個過程大約4年一次）。

  • Testnet

測試網路

具有與比特幣幾乎相同設計的獨立網絡，供開發人員測試改進。 ../../../assets/

  • Tor

洋蔥路由器

洋蔥路由器是可以保護用戶隱私的替代網路。

  • Txid

交易ID

交易ID是用於標記交易的獨特數字和字母字串。

  • UTXO

未使用的交易輸出

未使用的交易輸出是把一部分比特幣傳送到一個新的地址，作為未來交易的輸入使用。
